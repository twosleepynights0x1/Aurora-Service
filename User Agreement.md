1.1. Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между владельцем сервиса Aurora (далее — «Сервис», «Администрация») и физическим лицом (далее — «Пользователь»), использующим функционал Сервиса.

1.2. Сервис Aurora представляет собой программу лояльности, предназначенную для поощрения пользователей виртуальными бонусами. Сервис не является азартной игрой, лотереей, букмекерской конторой или иной деятельностью, связанной с риском утраты денежных средств.

1.3. Все взаимодействие с Сервисом осуществляется на безвозмездной основе. Сервис не принимает платежи, не осуществляет денежные переводы и не предоставляет услуги, связанные с реальными денежными средствами.

1.4. Акцептом настоящего Соглашения считается проставление отметки в поле «Я принимаю Пользовательское соглашение» при регистрации в Сервисе. Использование функционала Сервиса означает полное и безоговорочное принятие Пользователем всех условий настоящего Соглашения.

2.1. 2.2. Перечень обрабатываемых персональных данных включает в себя: номер мобильного телефона, никнейм (псевдоним), зашифрованный пароль, статистику использования Сервиса, историю действий, технические данные (IP-адрес, данные cookie).

2.3. Цель обработки персональных данных: предоставление доступа к функционалу Сервиса, идентификация Пользователя, ведение статистики, улучшение качества работы Сервиса, техническая поддержка, предотвращение мошеннических действий.
